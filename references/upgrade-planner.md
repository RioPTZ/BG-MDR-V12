# upgrade planner

Use this to decide what the quote engine should improve next.

## purpose
The engine should not grow randomly.
Use this planner to rank the next upgrades by value, friction removed, and impact on real quote work.

## how to use
- check current engine status first
- check current gaps
- map gaps to roadmap priorities
- prefer upgrades that reduce manual glue on real quote jobs

## current planning logic
1. remove manual price-resolution glue
2. connect image extraction to structured input more directly
3. reduce delivery friction
4. improve one-command flow from raw chat input
5. add regression fixtures so future upgrades stay safe
